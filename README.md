Data Visualization Python project using the Altair library to visualize a 25 year history of offensive football statstics in the NFL. The investigation aimed to correlate a general rise in offensive production over the last 25 years due partly to rule changes that favored offensive players by increasing penalties on defensive players. This the final project for CSCA 5702-Fundamentals of Data Visualization course at UC Boulder as part of the Masters in Computer Science program. Can we visualize an increase in penalties and offensive production simultaneously?
